## Description
Mechlink is a Django-based web application designed to provide on-road vehicle
breakdown assistance by connecting users with nearby mechanics.

## Features
- User and mechanic authentication
- Location-based mechanic allocation
- Live chat between users and mechanics
- E-commerce module for services and spare parts
- Responsive UI using Bootstrap

## Technologies Used
- Python, Django
- HTML, CSS, Bootstrap
- SQLite
- Google Maps API

## How to Run
1. Clone the repository
2. Install required packages
3. Run migrations
4. Start the Django server
